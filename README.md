# VIF

VIF is a fullstack mobile video-sharing app built with React Native (Expo) and Appwrite.
It lets users create an account, upload short videos, discover trending content, and manage their own profile posts.

## Tech Stack

- React Native + Expo Router
- NativeWind (Tailwind for React Native)
- Appwrite (Auth, Database, Storage)

## Notes

- Appwrite project configuration is defined in `lib/appwrite.js`.
- This project uses Expo, so you can run it with Expo Go or an emulator/simulator.
